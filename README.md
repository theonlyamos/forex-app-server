# Api

## Deployment

```
$ npm i
```

## Environment Variables

## Copy env variables of the env example

```
$ npm run cp:env
```

Must have set server and database variables

If you need in creating database then run command `$ npm run db:create`

## Migrations

```
$ npm run db:migrate
```

## Listen server

```
$ npm start
```

## Tests

```
$ npm run test
```
